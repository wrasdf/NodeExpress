NodeExpress
===========

Tec list:
1. Node
2. mongodb
3. Express
4. Ejs
5. less
6. rake
7. YUIcompressor

You can follow these steps to run this app:

1. git clone "this repro"
2. brew install mongodb
3. $ mongod
4. open a new shell tab run
  // development env
  $ rake run  
  // production env
  $ rake run ENV=prod
