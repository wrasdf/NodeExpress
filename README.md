Tec list:
>1. Node
2. mongodb
3. Express
4. mongoose
5. Ejs
6. less
7. rake
8. YUIcompressor

You can follow these steps to run this app:

1.clone code 
  <pre>git clone "this repro"</pre>
2. install mongodb app
  <pre>brew install mongodb</pre>
3. open mongodb app
  <pre>$ mongod</pre>
4. run different environment
<pre>
// development env
$ rake run
// production env
$ rake run ENV=prod</pre>
